# OkeyMate — Website

Landing page for the **OkeyMate** app, the companion app for the Okey tile game.

Live at: [okeymate.app](https://okeymate.app)

## About

OkeyMate helps you track scores, count rounds and roll dice during your Okey games.  
Available in German, Turkish and English.
